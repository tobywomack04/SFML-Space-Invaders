# SFML-Space-Invaders
<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/f8709411-a2eb-49e5-a960-075c8e5d0d40" />

Developed in C++ using SFML.<br>
You play as a spaceship who's job is to destroy the aliens before they reach you. Your goal is to survive for as long as possible and kill all the aliens.

Keybinds:
* Space (shoot)
* Up, Down, Left, Right arrows (movement)

Built using:
* C++
* Sfml - Simple and fast Multimedia Library
